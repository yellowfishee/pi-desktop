## 二、面板行为

### 2.1 侧边栏

| 状态 | 宽度 | 内容 |
|------|------|------|
| 展开 | 260px | 搜索框 + 会话列表 + 新建按钮 |
| 折叠 | 48px | 图标列表（hover 显示 tooltip） |
| 隐藏 | 0px | 完全隐藏（全屏消息面板） |

- 侧边栏宽度可拖拽调整（最小 200px，最大 400px）
- 折叠状态和宽度持久化到 AppConfig

### 2.2 属性面板

| 状态 | 宽度 | 触发方式 |
|------|------|---------|
| 收起 | 0px | 默认 |
| 展开 | 300px | 点击标题栏设置图标 / 点击消息气泡中的模型名称 |

- 属性面板宽度可拖拽调整（最小 240px，最大 420px）
- 展开状态和宽度持久化到 AppConfig

### 2.3 标题栏

自定义标题栏（Tauri `decorations: false`），包含：

```
[Pi Logo] [会话名] [模型标识 badge] ··· [侧边栏切换] [属性面板切换] [最小化] [最大化] [关闭]
```

- 整个标题栏为 drag region
- 模型标识 badge 显示当前模型简称（如 `Sonnet 4`），点击可快速切换
- 窗口控制按钮跟随系统风格

## 三、响应式策略

| 窗口宽度 | 侧边栏 | 属性面板 | 输入区域 |
|----------|--------|---------|---------|
| ≥ 1200px | 展开 | 可展开 | 正常 |
| 800-1200px | 可折叠 | 覆盖层 | 正常 |
| < 800px | 覆盖层 | 覆盖层 | 紧凑模式 |

覆盖层模式：面板浮在消息面板上方，点击外部关闭。

**交互**：

- 底部 "+" 按钮 → 新建会话

### 4.3 输入区域

```
┌──────────────────────────────────────────────────┐
│  [📎 附件]  输入消息... (Shift+Enter 换行)  [⏹] [➤]│
│             / 触发命令补全                          │

```

**交互**：
- Enter → 发送消息
- Shift+Enter → 换行
- `/` → 触发斜杠命令自动补全（Phase 2）
- 📎 附件按钮 → 文件选择器（图片）
- 拖拽图片到输入框 → 自动添加为附件
- Ctrl+V 粘贴图片 → 自动添加为附件
- AI 生成中 → 输入框禁用（Phase 1），显示 ⏹ 中止按钮
- 空内容时 → 发送按钮禁用

### 5.2 扩展对话框（extension_ui_request）

当扩展发起交互请求时，在消息面板底部弹出模态对话框：

```
┌──────────────────────────────────────────┐
│  🔒 权限请求                              │
│                                          │
│  Allow dangerous command?                 │
│                                          │
│  [Allow]  [Block]                        │
│                                          │
│         (10s 后自动选择默认值)             │
└──────────────────────────────────────────┘
```

支持的对话框类型：

| 方法 | UI 形式 | 响应 |
|------|--------|------|
| `select` | 列表选择弹窗 | `{ value: string }` 或 `{ cancelled: true }` |
| `confirm` | 确认/取消弹窗 | `{ confirmed: true/false }` 或 `{ cancelled: true }` |
| `input` | 文本输入弹窗 | `{ value: string }` 或 `{ cancelled: true }` |
| `editor` | 多行文本编辑弹窗 | `{ value: string }` 或 `{ cancelled: true }` |
| `notify` | Toast 通知 | 无需响应（fire-and-forget） |

### 5.3 通知系统（分级展示）

| 级别 | 形式 | 示例 |
|------|------|------|
| error | 模态弹窗 | pi 进程崩溃、API 错误 |
| warning | 黄色 Toast（右上角，5s 自动消失） | API Key 将过期、上下文接近上限 |
| info | 蓝色 Toast（3s 自动消失） | 会话已保存、模型已切换 |
| status | 底部状态栏文字 | "生成中…"、"重试中 (2/3)…" |
| inline | 消息流内系统消息 | 压缩摘要、分支摘要 |
