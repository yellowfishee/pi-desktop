# Step 4: 优化设置面板

## 任务
优化 SettingsPanel 中的字号设置体验，增加滑块控件和实时预览。

## 当前问题
- 只能输入像素值，没有直观的预览
- 预设按钮只有 [12, 14, 16, 18, 20]，缺少更细粒度
- 修改后没有实时预览效果

## 修复方案
1. 增加滑块控件（range input），范围 10-22px
2. 添加实时预览文本（显示 "这是预览文本 14px"）
3. 优化预设按钮布局，增加更多选项
4. 字号变化时实时应用到根元素

## 验收标准
- [ ] 设置面板有字号滑块
- [ ] 有实时预览文本
- [ ] 预设按钮包含 11, 12, 13, 14, 15, 16, 18, 20
- [ ] 修改字号后全局实时生效
