# Step 4.3: 扩展管理 UI

## 目标
提供扩展的查看、启用/禁用、安装/卸载界面，让用户管理 pi 扩展生态。

## 实施链路

### 1. 扩展列表获取
- 添加 `list_extensions` RPC 命令类型
- 调用 `sendCommand({ type: 'list_extensions' })` 获取扩展列表
- 扩展信息：name、version、enabled、description、author

### 2. ExtensionsPanel 组件
- 嵌入 SettingsPanel 新 tab 或独立面板
- 列表展示：名称 / 版本 / 状态（启用⚡ / 禁用🛑）
- 每个扩展旁有启用/禁用 toggle

### 3. 安装/卸载
- 添加 `install_extension` / `uninstall_extension` RPC 命令
- 安装：输入扩展名或路径
- 卸载：确认对话框

## 验收标准
- [ ] 扩展列表正确展示
- [ ] 启用/禁用 toggle 生效
- [ ] 安装/卸载功能可用
- [ ] 扩展状态变更后列表刷新
