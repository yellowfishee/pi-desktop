# Step 4.4: 全文搜索 + 上下文历史图表

## 目标
跨会话关键词搜索历史对话内容，并提供上下文使用的可视化历史图表。

## 实施链路

### 4.4a 全文搜索
- 添加 `search_sessions` RPC 命令：{ type: 'search_sessions', query: string }
- 返回匹配的会话列表 + 匹配片段
- 搜索结果 Modal：搜索输入框 + 结果列表
- 结果项显示：会话名 / 匹配片段（高亮关键词） / 时间
- 点击结果跳转到对应会话并定位到消息
- 快捷键：Cmd/Ctrl+Shift+F 打开搜索

### 4.4b 上下文历史图表
- 在 DiagnosticsPanel 或独立面板添加
- 记录每次 compaction 的时间点和压缩前后 token 数
- 用简易柱状图/折线图展示 token 使用趋势
- 从 `compaction_end` 事件中提取数据
- 存储最近 20 条 compaction 记录

## 验收标准
- [ ] 全文搜索可跨会话查找关键词
- [ ] 搜索结果高亮匹配片段
- [ ] 点击结果跳转到对应会话
- [ ] 上下文图表展示 token 使用趋势
- [ ] Compaction 事件自动记录到图表
