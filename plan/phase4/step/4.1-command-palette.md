# Step 4.1: 命令面板 (Cmd+P / Ctrl+P)

## 目标
仿 VS Code 命令面板，Cmd/Ctrl+P 弹出搜索框，快速切换模型、会话、执行命令。

## 实施链路

### 1. CommandPalette 组件
- Modal 浮层，Cmd/Ctrl+P 弹出，Esc 关闭
- 搜索输入框 + 实时过滤结果列表
- 支持键盘导航（↑↓ Enter）

### 2. 命令注册系统
- 定义 `CommandItem` 类型：{ id, label, category, icon?, action }
- 内置命令：
  - 切换模型（列出 availableModels）
  - 切换思考深度（列出 thinking levels）
  - 切换会话（列出最近会话）
  - 新建会话
  - 打开设置
  - Compact 压缩
  - Fork / Clone

### 3. 快捷键
- `Cmd/Ctrl+P` 打开命令面板
- `↑↓` 导航结果
- `Enter` 执行选中命令
- `Esc` 关闭

## 验收标准
- [ ] Cmd/Ctrl+P 打开命令面板
- [ ] 输入关键词实时过滤命令
- [ ] 可切换模型、思考深度、会话
- [ ] Esc 关闭，Enter 执行
