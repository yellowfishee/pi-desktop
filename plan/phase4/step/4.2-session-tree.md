# Step 4.2: 会话树导航

## 目标
可视化展示会话的分支/分叉结构，让用户直观看到 fork 产生的分支关系。

## 实施链路

### 1. 数据结构
- 从 pi 后端获取会话树结构（parent session → child forks）
- 每个节点包含：sessionId、sessionName、entryId、timestamp
- 树根为原始会话，分支为 fork 出的子会话

### 2. SessionTree 组件
- 树形视图组件，使用缩进 + 连接线
- 节点显示：摘要文本（第一条用户消息） + 时间 + 消息数
- 当前活跃会话高亮
- 点击节点切换到该分支
- 分支摘要气泡（branchSummary 消息）

### 3. Label 管理
- 右键节点可设置 Label（自定义名称）
- 调用 `set_session_name` 命令

### 4. 集成位置
- 侧边栏新增"分支"视图切换按钮
- 或在 PropertiesPanel 中展示当前会话的分支树

## 验收标准
- [ ] 可视化展示会话分支树
- [ ] 分支节点可点击切换
- [ ] 支持设置/编辑 Label
- [ ] 当前活跃分支高亮
