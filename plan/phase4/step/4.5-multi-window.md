# Step 4.5: 多窗口 / 多会话并行

## 目标
支持同时打开多个会话窗口，实现真正的多任务工作流。

## 实施链路

### 1. 技术评估
- Tauri 2 支持多窗口（WebviewWindow）
- 每个窗口独立运行 pi 进程（或共享单个 pi 进程）
- 方案 A：多 pi 进程（隔离性好，资源消耗大）
- 方案 B：单 pi 进程 + 多会话管理（高效，需 pi 支持）

### 2. 窗口管理
- 标题栏右键菜单："新建窗口"
- 快捷键：Cmd/Ctrl+Shift+N 新建窗口
- 每个窗口标题显示当前会话名
- 窗口列表在菜单栏 / Dock 中可见

### 3. 会话分配
- 新窗口默认从新会话开始
- 可从侧边栏拖拽会话到新窗口
- 窗口关闭时保存会话状态

### 4. 进程协调
- Rust 端维护 pi 进程池
- 每个窗口关联一个会话 → 一个 pi 进程
- 进程间通过 Tauri IPC 通信

## 验收标准
- [ ] Cmd/Ctrl+Shift+N 创建新窗口
- [ ] 多窗口独立运行不同会话
- [ ] 窗口标题显示会话名
- [ ] 窗口关闭时会话状态保存
