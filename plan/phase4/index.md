# Phase 4: 进阶

## 概述

Phase 1-3 完成了基础架构、功能完善、桌面体验。Phase 4 聚焦 v1.0 发布前的进阶功能，
按优先级排列：高频刚需 → 差异化特色 → 生态完善 → 锦上添花。

## 优先级

```
🔴 刚需            🟡 差异化          🟢 生态            ⚪ 锦上添花
4.1 命令面板  →  4.2 会话树导航  →  4.3 扩展管理  →  4.5 多窗口
4.4 全文搜索                          4.4 上下文图表
```

## 步骤链路

```
Step 4.1 → Step 4.2 → Step 4.3 → Step 4.4 → Step 4.5
  │           │           │           │           │
  │           │           │           │           └─ 多窗口/多会话
  │           │           │           └─ 全文搜索 + 上下文图表

  │           └─ 会话树导航（分支可视化）
  └─ 命令面板 (Cmd+P / Ctrl+P)
```
