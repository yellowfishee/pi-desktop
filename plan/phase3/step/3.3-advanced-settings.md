# Step 3.3: 高级设置面板

## 目标
在设置面板中增加"高级"tab，提供 bash、auto-retry、steering/follow-up 模式等控制。

## 实施链路

### 1. RPC 命令补充
添加以下命令到 RpcCommand：
- `BashCommand: { type: 'bash'; command: string }`
- `AbortBashCommand: { type: 'abort_bash' }`
- `SetSteeringModeCommand: { type: 'set_steering_mode'; mode: string }`
- `SetFollowUpModeCommand: { type: 'set_follow_up_mode'; mode: string }`
- `SetAutoRetryCommand: { type: 'set_auto_retry'; enabled: boolean }`
- `AbortRetryCommand: { type: 'abort_retry' }`

### 2. 高级设置 Tab
- SettingsPanel 新增"高级"tab
- Bash 执行：文本框 + 执行/中止按钮
- Auto-retry：开关 + 最大重试次数设置
- Steering mode：下拉选择（disabled/immediate/after_turn）
- Follow-up mode：下拉选择（disabled/immediate/after_agent）

### 3. SessionStore 状态
- 添加 `steeringMode`、`followUpMode`、`autoRetryEnabled` 状态
- 从 `get_state` 响应中初始化

## 验收标准
- [ ] 高级设置 tab 可用
- [ ] bash 命令可执行并显示结果
- [ ] auto-retry 开关正常工作
- [ ] steering/follow-up 模式设置生效
