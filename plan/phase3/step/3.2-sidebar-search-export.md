# Step 3.2: 侧边栏搜索过滤 + 导出 HTML

## 目标
在侧边栏顶部添加搜索输入框，实时过滤会话列表；右键菜单支持导出 HTML。

## 实施链路

### 1. 搜索过滤
- 侧边栏顶部添加搜索输入框（带搜索图标）
- 输入关键词 → 实时过滤 project 和 session name
- 匹配到的项目/会话高亮，不匹配的隐藏
- 空搜索词时显示全部

### 2. 导出 HTML
- 侧边栏右键菜单添加"导出 HTML"选项
- 需要先添加 `export_html` 命令到 RpcCommand
- 调用 `sendCommand({ type: 'export_html' })`
- pi 返回 HTML 文件路径，前端通过 Tauri dialog 选择保存位置
- 或者 pi 直接写入指定路径

### 3. RPC 类型补充
- `ExportHtmlCommand: { type: 'export_html'; outputPath?: string }`
- 加入 RpcCommand 联合类型

## 验收标准
- [ ] 搜索输入框实时过滤会话列表
- [ ] 匹配关键词的会话高亮，不匹配的折叠/隐藏
- [ ] 导出 HTML 命令可正常调用
- [ ] 右键菜单中显示"导出 HTML"选项
