# Step 5.4: 输入框增强

## 目标
输入框支持斜杠命令菜单、悬浮格式工具栏，以及接收扩展的 `set_editor_text` 预填充。

## 实施链路

### 1. 斜杠命令菜单（Step 5.1 已完成基础）
- 集成到 MessageInput，输入 `/` 触发
- 支持参数提示

### 2. 悬浮格式工具栏
- 输入框上方或内嵌显示格式按钮
- 支持：**加粗**、*斜体*、`代码`、> 引用
- 点击后插入 Markdown 语法并智能定位光标
- 也可用快捷键：Cmd+B / Cmd+I / Cmd+E

### 3. `set_editor_text` 支持
- 在 `pi-extension-ui-request` 事件处理中添加 `set_editor_text` 的处理
- 当扩展调用此方法时，将文本预填到输入框
- 需要 MessageInput 暴露 `setText` 方法或通过 store 通信
- 在 UIStore 中添加 `editorPrefill` 状态

## 验收标准
- [ ] 斜杠菜单在输入 `/` 时弹出
- [ ] 格式工具栏可用（加粗/斜体/代码/引用）
- [ ] 扩展可通过 set_editor_text 预填输入框
- [ ] 格式快捷键正常工作
