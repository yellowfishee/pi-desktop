# Step 5.1: 斜杠命令菜单

## 目标
用户在输入框输入 `/` 时，弹出斜杠命令列表，包含 pi 支持的所有命令和快捷操作。

## 实施链路

### 1. `get_commands` 命令
- 添加 `GetCommandsCommand: { type: 'get_commands' }` 到 RpcCommand
- 在输入 `/` 时调用，获取可用命令列表
- 命令结构：{ name, description, args? }

### 2. SlashMenu 组件
- 在输入框上方弹出浮层
- 显示命令列表：名称 + 描述
- 键盘导航（↑↓ Enter Esc）
- 选中后填充到输入框
- 继续输入参数

### 3. 内建命令
- `/compact` — 压缩上下文
- `/fork` — 从当前分叉
- `/clone` — 克隆会话
- `/model` — 切换模型
- `/thinking` — 切换思考深度
- `/new` — 新建会话
- `/export` — 导出 HTML
- 内置命令始终可用，即使 pi 的 `get_commands` 不可用

## 验收标准
- [ ] 输入 `/` 自动弹出命令菜单
- [ ] 显示从 pi 获取的命令 + 内建命令
- [ ] ↑↓ 导航，Enter 选中，Esc 关闭
- [ ] 选中后文本自动替换
