# Step 5.3: 消息操作右键菜单

## 目标
所有消息类型都有右键菜单，提供上下文操作。

## 实施链路

### 1. 用户消息右键菜单
- 编辑（已有双击，加右键入口）
- Fork from here
- 复制文本
- 删除（仅本地）

### 2. Assistant 消息右键菜单
- 复制整条回复
- 重新生成（发送相同的 prompt，带 `streamingBehavior`）
- 复制为 Markdown

### 3. 系统消息右键
- 复制摘要文本

### 4. 统一菜单组件
- 抽取 ContextMenu 通用组件（当前 Sidebar 中有独立实现）
- 支持 position、items 配置
- 复用创建，避免重复代码

## 验收标准
- [ ] 用户消息右键：编辑/Fork/复制/删除
- [ ] AI 消息右键：复制/重新生成/复制 Markdown
- [ ] 右键菜单定位准确，不超出视口
