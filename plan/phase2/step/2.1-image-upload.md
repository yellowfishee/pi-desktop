# Step 2.1: 图片上传 + 拖拽粘贴支持

## 目标
让用户可以在消息中附带图片发送给 pi，支持：
- 点击 "+" 按钮选择文件
- 拖拽图片到输入框
- 粘贴剪贴板图片（Ctrl+V）

## 实施链路

### 1. MessageInput 增强
- 添加图片预览区域（在 textarea 上方显示已选图片缩略图）
- "+" 按钮改为文件选择器（accept: image/*）
- 添加 drag-over 状态样式
- 添加 paste 事件监听（读取 clipboard items 中的 image 类型）
- 图片读取为 base64，存入本地 state `attachments: ImageContent[]`

### 2. 发送逻辑
- `sendCommand({ type: 'prompt', message, images })` 传递 images 数组
- 发送后清空 attachments

### 3. 消息气泡
- UserMessage 中渲染图片（`<img src="data:..." />`）
- 限制图片最大显示高度

## 验收标准
- [ ] 点击 "+" 可选择图片文件
- [ ] 拖拽图片到输入框显示预览
- [ ] Ctrl+V 粘贴剪贴板图片显示预览
- [ ] 发送消息时图片随消息一起发送
- [ ] 用户消息气泡中正确显示图片
- [ ] 发送后清空预览
