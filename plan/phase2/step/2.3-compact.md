# Step 2.3: Compact + Auto-compaction 控制

## 目标
让用户可以手动触发 compact，以及控制自动 compaction 开关。

## 实施链路

### 1. Compact 按钮
- 在 ChatPanel 或 StatusBar 添加 compact 按钮
- 调用 `sendCommand({ type: 'compact' })`
- 可选：弹窗让用户输入 customInstructions

### 2. Auto-compaction 开关
- Settings 面板 Agent tab 添加开关
- 调用 `sendCommand({ type: 'set_auto_compaction', enabled })`

### 3. Compaction 事件处理
- 监听 `compaction_start` → 设置 `isCompacting = true`
- 监听 `compaction_end` → 设置 `isCompacting = false`，显示 compaction 摘要
- 已有 `isCompacting` 状态在 sessionStore

### 4. Compaction 摘要气泡
- 在消息流中插入 compactionSummary 类型的消息
- 已有 `CompactionSummaryMessage` 类型定义
- MessageBubble 已有 compactionSummary 渲染（简要摘要卡片）

## 验收标准
- [ ] 手动 compact 按钮可用
- [ ] Settings 中可控制 auto-compaction 开关
- [ ] compaction 事件正确更新 isCompacting 状态
- [ ] Compact 完成后在消息流中显示摘要
